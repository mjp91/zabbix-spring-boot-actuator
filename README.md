# zabbix-spring-boot-actuator
Zabbix Template for Spring Boot Actuator Endpoints

## Items
 - Health
 - Version
 - JVM threads
 - JVM memory heap/nonheap
 - Process CPU Usage
 - Process uptime
 - Classes loaded
 - Internal server errors
 
 ## Graphs
 - JVM Memory (Pie chart heap vs nonheap)
 - JVM Memory (stacked)
 - JVM threads (stacked)
